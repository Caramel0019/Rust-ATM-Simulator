Akash Deployment API Documentation
Endpoints
1. Get Nonce
GET /auth/nonce
Retrieves a nonce for wallet authentication.
Request:

Headers: None
Parameters: None

Response (200 OK):
{
  "nonce": "<random_string>"
}

Errors:

429 Too Many Requests


2. Login
POST /auth/login
Authenticates a user and issues a session token.
Request:

Headers: Content-Type: application/json
Body:{
  "wallet_address": "<bech32_address>",
  "nonce": "<nonce>",
  "signature": "<signed_nonce>"
}



Response (200 OK):
{
  "token": "<session_token>",
  "expires_at": "<timestamp>"
}

Errors:

400 Bad Request
401 Unauthorized
429 Too Many Requests


3. Create Deployment
POST /deployments
Deploys a workload to the Akash Network.
Request:

Headers: Authorization: Bearer <token>, Content-Type: application/json
Body:{
  "sdl": "<SDL_config>",
  "dseq": "<deployment_sequence>",
  "parameters": {}
}



Response (201 Created):
{
  "deployment_id": "<dseq>",
  "status": "pending",
  "created_at": "<timestamp>"
}

Errors:

400 Bad Request
401 Unauthorized
429 Too Many Requests


4. Get Deployment Status
GET /deployments/{id}
Retrieves a deployment's status.
Request:

Headers: Authorization: Bearer <token>
Path: id (string, deployment ID)

Response (200 OK):
{
  "deployment_id": "<dseq>",
  "status": "<active|pending|terminated>",
  "resources": {}
}

Errors:

401 Unauthorized
404 Not Found
429 Too Many Requests


5. Terminate Deployment
DELETE /deployments/{id}
Terminates a deployment.
Request:

Headers: Authorization: Bearer <token>
Path: id (string, deployment ID)

Response (204 No Content):

No body

Errors:

401 Unauthorized
404 Not Found
429 Too Many Requests


6. List Deployments
GET /deployments
Lists all deployments for the authenticated user.
Request:

Headers: Authorization: Bearer <token>

Response (200 OK):
[
  {
    "deployment_id": "<dseq>",
    "status": "<active|pending|terminated>",
    "created_at": "<timestamp>"
  }
]

Errors:

401 Unauthorized
429 Too Many Requests


7. Get Resource Usage
GET /deployments/{id}/resources
Retrieves resource usage for a deployment.
Request:

Headers: Authorization: Bearer <token>
Path: id (string, deployment ID)

Response (200 OK):
{
  "deployment_id": "<dseq>",
  "cpu": "<millicores>",
  "memory": "<bytes>",
  "storage": "<bytes>"
}

Errors:

401 Unauthorized
404 Not Found
429 Too Many Requests


8. Process Voice Command
POST /voice/process
Processes an audio file for voice commands.
Request:

Headers: Authorization: Bearer <token>, Content-Type: multipart/form-data
Body:
audio (file, e.g., WAV, MP3)
command_type (string, optional)



Response (200 OK):
{
  "command": "<parsed_command>",
  "confidence": <float>,
  "processed_at": "<timestamp>"
}

Errors:

400 Bad Request
401 Unauthorized
413 Request Entity Too Large
429 Too Many Requests
